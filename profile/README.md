# Fission — Portable Lossless Audio Editing with Direct Export

## About Fission

Fission provides a complete lossless audio editing environment for users who need to edit without quality loss. Fission combines cut, copy, paste, split, trim, and crop — all without re‑encoding. Fission enables users to edit MP3, AAC, FLAC, WAV, AIFF, and Apple Lossless directly. Fission runs on desktop platforms. Fission is trusted by podcasters, voiceover artists, audiobook editors, and audio professionals worldwide.
